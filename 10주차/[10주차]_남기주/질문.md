1. 요소 노드 취득 방법중에 클래스와 아이디를 사용한 취득 방법을 설명하시오.

2. 다음 코드를

```html
<!DOCTYPE html>
<head>
  <style>
    .red {
      color: red;
    }
    .blue {
      color: blue;
    }
  </style>
</head>
<html>
  <body>
    <ul id="fruits">
      <li class="red">Apple</li>
      <li class="red">Banana</li>
      <li class="red">Orange</li>
    </ul>
    <script>
      const $elems = document.getElementsByClassName("red");
      for (let i = 0; i < $elems.length; i++) {
        $elems[i].className = "blue";
      }
    </script>
    <body>
      <html></html>
    </body>
  </body>
</html>
```
